# 采样与评估流程说明

按超参数执行顺序组织的配置与脚本流程。

## 配置与脚本执行顺序

| 阶段 | 配置块 / 脚本 | 说明 |
|------|---------------|------|
| **0** | `model` | 模型加载：checkpoint、grad_fusion 等 |
| **1** | `sample` 顶层 | 全局参数：seed、mode、num_samples、num_steps、sample_num_atoms、center_pos_mode、pos_only |
| **2a** | `sample.dynamic` | dynamic 模式：large_step → selector（可选）→ refine |
| **2b** | `sample.targetdiff_baseline_refine` | TargetDiff 基准修复（可选）；未开优化时在 dynamic 后执行；`optimization.enable=true` 时在优化结束后对全部输出分子执行 |
| **2c** | `sample.optimization` | 优化模式（`after_dynamic=true` 时先 dynamic 再优化；与 2b 同时开启时 2b 在优化之后） |
| **3** | `sample.pre_docking` | 对接前预处理 |
| **3** | `sample.step_trajectory_save` | 跳步留存（可视化） |
| **4** | `evaluate_pt_with_correct_reconstruct.py` | 从 .pt 结果重建分子、对接、计算指标（JSD、Vina 等） |
| **4c** | `evaluate_pocket_quality.py` | 口袋质量评估 |

## 脚本路径

- 阶段 4c：`/home/user/Desktop/Ye/DiffDynamic/evaluate_pocket_quality.py`
